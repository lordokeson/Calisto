Logs Management
Users can aggregate, search, and analyze logs efficiently using advanced log management tools. These tools offer real-time monitoring, allowing users to view log data as it is generated. Security Event Management
The platform includes robust security event management features. It provides threat detection mechanisms to identify potential security breaches, compliance tracking to ensure adherence to industry standards, and real-time alerts to notify users of suspicious activities immediately. These features help maintain a secure environment and minimize risks.
Metrics Management
Our service offers comprehensive metrics management to track application performance, resource usage, and other critical metrics. Users can set up dashboards to visualize key performance indicators (KPIs) and receive alerts when metrics exceed predefined thresholds. This helps in proactive performance tuning and resource optimization, ensuring smooth operation and high availability of applications.
Datahub Warehouse Connection
The platform supports seamless integration with various storage platforms through its Datahub Warehouse Connection feature. This enables users to connect and streamline data across different storage solutions, facilitating easier access and better data integration. With this feature, users can manage their data more effectively, ensuring that it is readily available for analysis and reporting.

"type" is a logical field used to differentiate between logs (if you're coming from using base Elasticsearch, you might know these as "documents"). in Calisto, "type" also acts as the main condition for log parsing. Ideally, you want to set the type based on log format and not log source. For example, If you have five servers sending Apache Access logs, they should all be shipped under the "apache_access" type, with an additional parameter to indicate the logs' environment.

Calisto supports a variety of shipping methods to cater to the different ways you access your logs: If your logs are written to file, we strongly recommend using Filebeat - it's lightweight, reliable and allows for quite a bit of freedom when it comes to configuration. If you want to ship logs directly from your application, we offer appenders for multiple programming languages that you can slot into your existing code and ship directly from there.
